# 3D-Game
Project 5 for MSCH-220
Dec 10, 2023
A 3D FPS game that is lightly inspired by Among Us.

## Implementation
Godot 4

## What Makes this a Game?
This is a game because the player is able to control their character and there are challenges to overcome.
